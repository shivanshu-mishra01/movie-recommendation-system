\# 🎬 Movie Recommendation System



A machine learning based web application that recommends movies similar to the selected movie.



\## 🚀 Features

\- Recommend similar movies

\- Shows movie posters

\- Clean and interactive UI

\- Fast recommendations



\## 🛠 Tech Stack

\- Python

\- Pandas

\- Scikit-learn

\- Streamlit

\- Requests API



\## 📂 Files Included

\- app.py

\- movie\_dict.pkl

\- similarity.pkl

\- requirements.txt



\## ▶️ Run Locally



pip install -r requirements.txt



streamlit run app.py
